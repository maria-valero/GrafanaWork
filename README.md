# GrafanaWork

This scripts copy a dashboard in grafana, made a modification in their JSON file, and then import again the dashboard.

## Usage

### export.sh

./exporth

### modify.py

python modify.py [NEW_MAC] [DASHBOARD_TO_COPY]

### import.sh

./import [DASHBOARD_NAME]
